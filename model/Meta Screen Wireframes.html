Titre "Meta"

Sentence case, sobre. Pas de sous-titre — la donnée parle d'elle-même. Hauteur ligne tight pour ancrer le header.

--fs-2xl weight 700 --tr-tight --fg1
2

Fraîcheur des données

Icône clock + texte tabulaire. Couleur --fg3 pour rester discret — la donnée seule importe. Tap : tooltip avec source + horaire de pull.

--fg3 --fs-sm tabular-nums
3

Segmented control Doubles / Singles

Largeur 100 %. Variante accent — le segment actif passe en --accent car c'est le switch principal de l'écran. Persiste entre sessions (AsyncStorage). Au switch : tout le contenu se recharge avec Shimmer.

--accent --surface-sunken --r-pill --fg-inverse Reacticx · Segmented Control
4

Bande de tendances

3-4 chips informatifs, non-cliquables, scroll horizontal libre. Icône Ionicons en couleur sémantique (success / danger / warning) — la valeur Pokémon reste en --accent. Disparaît proprement sur les bords (pas de fondu — coupure nette acceptable).

--surface --border --r-pill padding 6 · 11 --fs-sm
5

Tabs Usage / Winrate / Cores

Underline accent sur l'onglet actif (pas de pill). Inactifs en --fg3 — pas de texture, pas de fond. Onglet Winrate possiblement gated (lock icon) selon les données disponibles.

--accent --fg1 / --fg3 border-bottom 2px Reacticx · Tabs
6

Sprite + nom + types

Sprite 40×40 sur fond teinté du type primaire (--t-{type}-bg). Filtre silhouette appliqué — pas de sticker art coloré (cf. DS § Imagery). Nom en weight 500 ; 2 badges de type max sous le nom, casing sentence.

--t-fire / --t-water / … --t-fire-bg / --t-water-bg / … --r-pill --r-sm filter: brightness(0) invert
7

Barre d'usage + %

Barre relative au #1 (rang 1 = 100 % largeur). Hauteur 3pt — fine, presque graphique. % en tabular-nums espace insécable avant le signe. Animation à l'apparition via Reacticx · Rolling Counter.

--accent --surface-high (rail) --r-pill --fs-sm weight 600 Progress
8

Trend W/W

Icône seule, pas de chiffre — la magnitude est dans la barre. success pour hausse, danger pour baisse, --fg-faint pour stable. Hit-slop élargi (icône 16pt, target 44pt).

--success --danger --fg-faint (stable) trending-up / -down / minus
·

Séparateur de row

Hairline 1px --border avec inset gauche aligné sur le nom (76pt depuis le bord). Pas de divider entre rank et sprite — l'œil suit la colonne.

Contraintes UX — invariantes de l'écran

  • Header fixe (titre + segmented) — accessible en 1 tap depuis tout état
  • Switch Doubles/Singles persiste entre sessions (AsyncStorage)
  • Liste 20 items en FlashList — pas FlatList (perf scroll)
  • Modal scrollable indépendamment de la liste sous-jacente
  • Toutes les sprites en lazy-load avec shimmer placeholder
  • Touch target ≥ 44×44pt partout (rows 64pt, icônes hit-slop élargi)
  • Animation à l'apparition des % via Rolling Counter — effet premium
  • Refresh auto toutes les 30 min quand l'app est foreground
  • Pas de spinner — shimmer partout
  • Pas d'emoji dans la copy ni dans les badges
  • Pas de gradient coloré — radial wash accent autorisé sur hero uniquement
  • Pas de stat coloré par-défaut — accent réservé aux interactions clés
Reacticx · Segmented Control
2 instances : Doubles/Singles dans le header (variante accent) + dans le modal (variante neutre).
Reacticx · Tabs
3 onglets : Usage / Winrate / Cores. Underline accent qui glisse. Winrate gated par disponibilité de données.
Reacticx · Progress
Barres d'usage (liste), barres de stats (modal), barres de fréquence (moves/items). Variantes de couleur sémantique.
Reacticx · Shimmer
Tous états de chargement — rows, hero, sections. Animation opacity 0.35 → 0.65 sur --surface-high.
Reacticx · Rolling Counter
Animation des % à l'apparition (240ms standard). Effet premium signature, jamais sur cachage / shimmer.
Reacticx · Accordion
Optionnel — sections Counters et Speed tiers si l'espace manque. Header tappable, contenu animé en hauteur 240ms decel.
