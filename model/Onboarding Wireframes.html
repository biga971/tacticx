Contraintes UX — invariantes du flow

  • Une seule question par écran — jamais deux interactions en parallèle
  • CTA toujours visible sans scroll, sur tous les viewports ≥ 320pt
  • 3 cards + CTA + header tiennent sans scroll sur iPhone SE
  • Bouton désactivé tant qu'aucune option n'est sélectionnée — pas de skip
  • Back possible à chaque étape sauf la première — geste swipe-right inclus
  • Toutes les cards sont dans la zone pouce (66 % bas de l'écran)
  • Friction zéro — pas de compte, pas d'email, pas de mot de passe
  • Dark mode uniquement (MVP) — pas de variant light
  • Pas d'image, pas d'illustration — icônes seulement
  • Pas d'emoji dans la copy — ni dans les badges, ni dans les empty states
  • Pas de Title Case — sentence case partout
  • Aucune mention des noms de jeux (Pokémon, Showdown, etc.) en UI
