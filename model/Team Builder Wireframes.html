Titre + segmented filtre

Titre "Mes équipes" sentence case. Le segmented VGC / Singles filtre la liste en temps réel (n'affiche que les équipes du format actif). Variante accent — c'est le switch principal de l'écran.

Bouton créer

Icône add-circle-outline en accent, haut-droit. Tap → router.push('/teams/builder') avec un nouveau Team vierge pré-rempli au format actif.

--accent34×34 (hit 44)

6 mini-sprites

Row de 6 sprites 32pt teintés par type. Chargés via expo-image (cache persistant) + Shimmer au mount. Slot vide = cercle --surface + help-outline en --fg-faint.

--surface-high--t-{type}-bg--r-pillShimmer

Nom + badges + date

Nom éditable inline (tap → champ texte). Badge format --accent-soft (VGC/Singles) via Badge, badge style optionnel neutre (Trick Room / HO / Balance / Stall). Date relative --fg3 tabulaire à droite.

--accent-soft--surface-high--r-sm--fg3 (date)
5

Swipe droit → dupliquer

Révèle copy-outline sur fond --info. Action immédiate sans confirmation (non-destructive). La copie est ajoutée en tête de liste avec suffixe "(copie)".

--info--fg-inverse (icône)swipe-to-action

Équipe partiellement remplie

Une équipe peut être sauvegardée incomplète (slots vides montrés en help-circle). Le builder n'impose pas 6 Pokémon — l'utilisateur travaille de façon itérative.

--surface (slot vide)--fg-faint

Swipe gauche → supprimer

Révèle trash-outline sur fond --danger. Destructive → confirmation via Toast avec action "Annuler" (5 s) avant suppression définitive. Pas de dialog modal bloquant.

--danger--fg-inverseToast

Header sticky — nom éditable

Champ nom avec pencil-outline à gauche, tap → éditable inline (underline dashed indique l'affordance). options-outline à droite → Dropdown : Importer Pokepaste / Exporter / Dupliquer / Supprimer.

--fg1--fg3 (pencil)--border-strong (underline)Dropdown

Grille 2 colonnes (VGC)

2 colonnes × 3 lignes en VGC ; bascule en 1 colonne × 6 en Singles selon le segmented. Slot rempli : --surface + --r-lg, sprite 64pt, nom + TypeBadges XS. Long-press → mode réorg (scale 1.03 + shadow, drag & drop).

--surface--border--r-lglong-press → reorder
3

Overlays de slot

create-outline haut-droit (tap → ouvre PokemonEditorSheet) ; close-outline haut-gauche (retirer le slot, confirmation inline). Tuiles circulaires --surface-sunken pour ressortir sur le sprite.

--surface-sunken--border--fg2hit-slop élargi
4

Tabs Défensif / Offensif / Speed

Underline accent. Le panel d'analyse est sticky-scrollable sous la grille. Tabs. Switch d'onglet : cross-fade du contenu (160ms). Le header du tableau reste sticky lors du scroll interne.

--accent--fg1 / --fg3Tabs
5

Matrice — cellules colorées

Cellules 36×36 --r-sm, texte 11px bold centré. Code couleur du multiplicateur reçu :

4× --danger-soft + border (faiblesse forte) 2× --warning-soft (faible) · --surface (neutre 1×) ½ --success-soft (résiste) IMM --accent-soft (immunité 0×)

Recalcul à chaque changement de slot ; cellules modifiées font un fade ancienne→nouvelle valeur. Calculé offline (table 18×18, dual-types).

lib/calc/typeCoverage.tsheader sticky-x
6

Colonne Score + tri danger

Score = nb de Pokémon en ×2/×4 sur ce type. Badge rouge si ≥ 4, orange si ≥ 2, gris si 0. Tri par défaut : lignes les plus exposées en premier. Si ≥ 4 faibles au même type → ligne fond rouge + flag DANGER. Si 0 résiste → badge orange "Non couverte".

--danger-soft (≥4)--warning-soft (≥2)--surface (0)
7

Footer sticky — 4 actions

Respecte la safe area basse. --accent Sauvegarder (primaire) + 3 ghost : Pokepaste (export → share sheet + Toast "Copié"), Calc, Budget VP (modal récap avec Progress par Pokémon + badge Économique/Standard/Coûteux).

--accent--fg2 (ghost)ToastProgress

Affichée sous le nom. Stat boostée en +Vit accent, diminuée en −Atq danger. Ce codage couleur se propage aux labels et valeurs finales des sliders concernés.

--accent (+stat)--danger (−stat)--fg3

Talent via Dropdown ; Objet via Picker (sprite item 24pt + nom + recherche inline) ; Nature via Dropdown (25 natures avec indicateurs +/−). 4 slots de moves, tap → search overlay filtrable (type/catégorie/puissance) en FlashList.

--surface--border--r-mdDropdownPicker

Compteur EV — X / 508

Compteur dynamique dans l'eyebrow. Passe en --warning proche de 508, --danger si dépassé. Tabulaire. Ici 340/508 (warning car > 2 stats max).

--warning--dangertabular-nums

EV sliders — snap ×4

Range 0→252, snap aux multiples de 4 (haptic légère via expo-haptics). Thumb 18pt (touch 44pt). Track rempli --accent. Stat boostée nature → label + valeur finale en --accent (Vit) ; diminuée → --danger (Atq, slider disabled à 0). Valeur finale Rolling Counter.

--accent (fill/thumb)--surface-sunken (track)--r-pillRolling Counter
6

Barre de budget EV

Progress horizontal sous les 6 sliders. "X EV restants" — --success si > 60, --warning 12-60, --danger < 12. Fill animé en temps réel pendant le glissement. Input numérique à droite de chaque slider clamp 0-252 + arrondi ×4 + Toast si dépasse le budget.

--success--warning--dangerProgressToast
7

Presets + IVs — Accordion

"Spreads populaires" (top 3 méta depuis l'API, Shimmer au fetch). "Appliquer" → sliders s'animent vers les nouvelles valeurs (spring 400ms). "IVs (avancé)" masqué par défaut, 6 inputs 0-31 (31 par défaut) + note contextuelle "IVs 0 en Attaque" si applicable.

--surface--accent-soft (apply)AccordionShimmer

Contraintes UX — invariantes du Team Builder

  • FlashList pour toutes les listes (équipes, résultats search, moves)
  • Segmented VGC/Singles dans chaque header — bascule la grille 2×3 ↔ 1×6
  • Matrice recalculée offline à chaque changement de slot
  • Sliders EV : snap ×4 + haptic + touch target 44×44pt sur le thumb
  • Stat boostée/diminuée par nature codée couleur partout (accent/danger)
  • Compteur EV 508 + budget restant en temps réel
  • Footer + header sticky, safe area haut ET bas
  • Long-press slot → réorganisation drag & drop
  • Pas d'emoji, pas de SF Symbol — Ionicons -outline uniquement
  • Pas de spinner — Shimmer partout
  • Premium (historique winrate, sync) : visible + flou + lock, jamais caché
  • Pas de valeur en dur — tokens theme.ts uniquement
